# アプリの概要
再生リストの動画の中から見たい動画を検索することができます。  
検索方法としてはキーワードを入力して、概要欄に入力したキーワードの文字列が含まれる動画が出力されます。     

  --

# 使用方法
キーワードと再生リストidを入力して動画表示ボタンをクリックします。
